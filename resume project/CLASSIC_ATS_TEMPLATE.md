# 📋 Classic ATS Template - Documentation

## Overview

The **Classic ATS Template** is designed for **maximum ATS (Applicant Tracking System) compatibility** with a **100% parsability score**. This template follows industry-standard formatting that is easily read by all ATS software used by companies worldwide.

---

## ✅ **ATS Score: 100%**

### Why This Template Gets Selected More Often:

1. **Simple, Linear Layout**
   - No complex columns or tables
   - Left-aligned text throughout
   - Clear section hierarchy
   - Standard font (serif - Times New Roman style)

2. **Standard Section Headers**
   - Professional Summary
   - Technologies
   - Education
   - Experience
   - Projects
   - Achievements
   - Certifications

3. **Clean Formatting**
   - Simple bullet points (•)
   - No graphics or images
   - No colored backgrounds
   - Minimal borders (only for headers)
   - Standard spacing

4. **Contact Information**
   - Clear, centered header
   - Email, phone, LinkedIn, GitHub in standard format
   - Pipe separators (|) between items

## 📊 **Template Structure:**

### 1. **Header Section**
```
FULL NAME (Large, Bold, Centered, Uppercase)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
email@domain.com
Phone | LinkedIn | GitHub
Location
```

### 2. **Professional Summary**
- **Format**: Single paragraph, text-justified
- **Length**: 3-5 sentences
- **Content**: Technical focus, skills, passion, goals
- **Keywords**: Industry-specific terms for ATS matching

### 3. **Technologies Section**
- **Categories**: Languages, Tools, Frameworks
- **Format**: Category: item1, item2, item3.
- **Why**: Easy for ATS to extract skill keywords

### 4. **Education**
```
Institution Name, Location — Degree
Start Date – End Date
GPA: X.X/10.0
Coursework: Course 1, Course 2, Course 3.
```

### 5. **Experience** (if applicable)
```
Company Name
Position Title
Location
Start Date – End Date

• Achievement with metrics
• Another achievement with impact
• Technical contribution
```

### 6. **Projects**
```
Project Title                    Github
• Key feature or achievement
• Technical implementation detail
• Impact or result
Tools Used: Tech1, Tech2, Tech3.
Context: Competition/hackathon details.
```

### 7. **Achievements**
```
Competition/Award Name, Organization
Description of achievement and recognition.
```

### 8. **Certifications** (if applicable)
```
• Certification Name - Issuer (Date)
• Another Certification - Issuer (Date)
```

## 🔧 **How to Use:**

### In the App:

1. **Create Resume**
   - Choose AI Generation or Manual Form
   - Enter your details

2. **Select Template**
   - Choose "Classic ATS"
   - Preview your resume

3. **Download**
   - Download as PDF
   - Print-ready format
   - ATS-optimized!

### Tips for Maximum Impact:

#### ✅ **Do:**
- Use action verbs (Developed, Implemented, Designed)
- Include metrics and numbers (5 years, 10+ projects)
- Add links (Github, LinkedIn, Portfolio)
- List specific technologies
- Mention competitions/achievements
- Use industry keywords
- Keep it to 1-2 pages

#### ❌ **Don't:**
- Use pronouns (I, me, my)
- Add photos
- Use fancy fonts
- Include tables
- Use text boxes
- Add unnecessary colors
- Exceed 2 pages
